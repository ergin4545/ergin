# ERGİN

ERGİN, Bluetooth tabanlı tarama uygulamasıdır.

## Özellikler

- Bluetooth cihaz bağlantısı
- Taramayı başlatma ve durdurma
- 5x5 tarama ekranı
- Tarama verilerini yerel olarak kaydetme
- Ayarları saklama
- PWA desteği (Ana ekrana yüklenebilir)

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- Web Bluetooth API
- LocalStorage
- Progressive Web App (PWA)

## Dosyalar

- index.html
- style.css
- app.js
- bluetooth.js
- scan.js
- storage.js
- settings.js
- manifest.json

## Kurulum

1. Depoyu indir veya klonla.
2. Dosyaları HTTPS üzerinden yayınla (GitHub Pages).
3. Google Chrome ile aç.
4. Bluetooth cihazına bağlan.
5. Taramayı başlat.
