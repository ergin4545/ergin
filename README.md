# Yer Altı Görüntüleme Uygulaması

Bu proje yer altı tarama, sensör bağlantısı ve veri görüntüleme amaçlı hazırlanmıştır.

## Dosyalar
- index.html
- style.css
- app.js
- bluetooth.js
- scan.js
- settings.js
- storage.js

## Özellikler
- Bluetooth bağlantısı
- Sensör tarama altyapısı
- Ayarlar bölümü
- Veri depolama
- Mobil uyumlu arayüz

## Teknolojiler
- HTML
- CSS
- JavaScript
